CÓMO SUBIRLA A HOSTINGER
1. Entrá al panel de Hostinger.
2. Abrí Administrador de archivos.
3. Entrá en la carpeta public_html.
4. Eliminá el archivo index.html de ejemplo si existe.
5. Subí todos los archivos de esta carpeta, manteniendo la estructura.
6. Verificá que index.html quede directamente dentro de public_html.
7. Abrí tu dominio.

FUNCIONES INCLUIDAS

IMPORTANTE
Esta es una versión frontend estática. El carrito no cobra ni procesa pagos.
Para vender online de forma completa todavía hay que integrar:
- pasarela de pago,
- stock,
- panel administrador,
- base de datos,
- envíos,
- usuarios y pedidos.
